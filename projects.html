Designed and implemented a C++ program using hashing and binary search trees to emulate a basic relational database with an interface based on a subset of a standard query language. The project involved selecting appropriate data structures for optimal performance, evaluating run-time and storage trade-offs for storing and accessing data contained in multiple data structures, and designing algorithms to handle various situations.

Implemented Minimum Spanning Tree and Branch and Bound algorithms in C++ to solve the Travelling Salesperson Problem on a graph represented by verticies and edges, and evaluated the efficiency of Prim’s Algorithm and Kruskal’s Algorithm for different scenarios. Additionally, I implemented various TSP heuristics and developed a fast and effective bounding algorithm to achieve a nearly-optimal TSP solution.

Developed a C++ program for solving a 3D maze using advanced graph traversal algorithms such as breadth first search (BFS) and depth first search (DFS) with optimized memory management and data structures such as FIFO queue and LIFO stack data structures.

Created an AI and human player game that can be mixed and matched which plays a complex card game at a human-comparable level, demonstrating in-depth knowledge of polymorphism, C++ abstraction, and human-computer interfacing.

Created a program that uses seam carving for content-aware resizing of images. The algorithm works by finding and removing “seams” in the image that pass through the least important pixels demonstrating knowledge in C style pointers, Streams I/O, and Abstract Data Types.

Utilizing C++ and priority queues, I developed a sophisticated stock market simulation that facilitates seamless trades between buyers and sellers. Not only does the program calculate a real-time running median, but it also identifies the most opportune moment to buy and sell a given commodity, maximizing potential profit margins. The result is an advanced trading tool that accurately replicates the complexities of real-world market fluctuations.
